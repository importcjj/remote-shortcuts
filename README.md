# remote-shortcuts
通过手机使用快捷键操作Windows程序

# 使用说明

1. 手机和电脑处于同一局域网
2. 电脑端启动本程序
3. 手机扫码进入管理界面
4. 打开需要控制的程序

# 问题

1. 该版本还很简单，目前是我用于躺在床上控制爱奇艺客户端
2. 被控制的客户端需要处于激活状态
